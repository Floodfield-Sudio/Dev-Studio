DevStudio Pro

Python IDE + Minecraft mod builder — all-in-one, no administrator rights required.

Features

✨ Features

🐍 Python mode

  • Multi-tab editor with full syntax highlighting (keywords, builtins, docstrings, f-strings…)
  • Line numbers, folding of class/def, current line highlighting
  • Automatic pip dependency injection on save
  • Build .exe / Linux/macOS binary via PyInstaller
  • Dev → stable version system
  • Automatic updates for the published app (via GitHub Releases) - in development

⛏ Minecraft mode

  • Supported loaders: Forge, NeoForge, Fabric, Quilt
  • Versions: 1.7.10, 1.12.2, 1.16.5, 1.18.2, 1.19.x, 1.20.x, 1.21.x
  • JDK 8 / 17 / 21 downloaded automatically (no admin rights)
  • Build gradlew build with real-time logs
  • Automatic Gradle wrapper injection from the cached MDK

🎮 Minecraft instances

  • Isolated Vanilla / Forge / NeoForge / Fabric / Quilt instances
  • Mode FULL OFFLINE mode — no Mojang account required
  • Offline single-player launch
  • Dedicated local server (LAN / multiplayer, online-mode=false)
  • Automatic copy of the compiled mod into each instance
  • Built-in server terminal (commands /op, /say, andc.)

Installation & quick start

🚀 Installation

Windows

  1. Download Python 3.10+ — check "Add Python to PATH"
  2. Download DevStudioPro.pyw and run.bat from the Releases
  3. Place both files in the same folder
  4. Double-click run.bat

The app installs to %APPDATA%\FFS\DevStudio\, PyQt6 is installed automatically.

macOS / Linux

# Download DevStudioPro.pyw and run.sh
chmod +x run.sh
./run.sh

Quick start

Python project

  1. 📂 Open project → select the folder
  2. Edit files (double-click in the file explorer)
  3. ▶ Run (F5) to test
  4. Tab 🐍 Python → fill in name, version, repo
  5. 🔧 Build DEV → generates the exe + _updater.py + installers

Minecraft mod

  1. Open the mod folder (dandected if gradlew is present)
  2. Tab ⛏ Minecraft → choose loader + MC version
  3. Download the JDK if needed (auto button)
  4. 🔨 Build → build, real-time logs
  5. Tab 🎮 Instances → create, install, launch
