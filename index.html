🛠 DevStudio Pro

IDE Python + Builder de mods Minecraft — sans droits administrateur.

✨ Fonctionnalités Clés

🐍 Mode Python

  • Éditeur multi-onglets + Coloration complète
  • Auto-injection des dépendances pip
  • Build .exe via PyInstaller intégré
  • Génération d'installeurs graphiques .pyw

⛏ Mode Minecraft

  • Forge, NeoForge, Fabric, Quilt
  • JDK 8/17/21 portable (auto-téléchargé)
  • Instances offline (pas de compte requis)
  • Serveur LAN local avec terminal intégré

📥 Installation

Windows

# 1. Installer Python 3.10+ (Add to PATH)
# 2. Télécharger DevStudioPro.pyw et run.bat
# 3. Lancer run.bat

macOS / Linux

chmod +x run.sh && ./run.sh
